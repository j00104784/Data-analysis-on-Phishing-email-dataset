## Phishing Email Data Analysis

### Overview
This project analyzes a phishing email dataset (CEAS_08.csv) using Pandas, Seaborn, and WordCloud.

#### Tasks Performed
- Data loading and inspection
- Cleaning and preprocessing
- Visualization of class distribution
- Word cloud generation for phishing and legitimate emails

#### Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib
- WordCloud

#### Dataset
CEAS_08.csv (Kaggle phishing dataset)  
Dataset obtained via *Al-Subaiey, A., Al-Thani, M., Alam, N. A., Antora, K. F., Khandakar, A., & Zaman, S. A. U. (2024, May 19). Novel Interpretable and Robust Web-based AI Platform for Phishing Email Detection. ArXiv.org. https://arxiv.org/abs/2405.11619*  
Download from Kaggle:  https://www.kaggle.com/datasets/naserabdullahalam/phishing-email-dataset/data
